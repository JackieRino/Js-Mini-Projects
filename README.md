# Js-Mini-Projects
A collection of small JavaScript Projects including a counter, score tracker, temperature tracker, event listener demos to practice core Javascripst concepts, and a students stats dashboard in progress..


These projects were built while learning JavaScript fundamentals. The goal was to reinforce core concepts through repitition and experimentation.
Concepts Covered: #Variables and State #Dom selection and manipulation #Event Listeners #Functions and Scope #Conditional Logic #Updating the Ui based on user interaction
Projects:

Counter
increment and decrement a value.

Score Tracker 
Update score dynamically using event listeners.

Temperature Tracker 
Updates console based on conditions.

Mouseover Event 
Demo Experiment with mouseover and other event listeners to understand browser events.

Student Stats Dashboard
A javascript-based data handling project that mananges a list of students using an array of objects. Each student contains a first nmae, last name and a score, which is dynamically rendered to the page. It calculates basic statistics and includes basic CSS sytling for layout and presentation. 
This project is still in progress with planned imporvements such as an imporved UI design.
